Title: Randomized Optimization

Author: Richard HUANG

## Instructions

- Download this folder from https://github.com/Richuo/cs7641/tree/master/hw2 (Dataset already included)

- python setup.py install (to install custom mlrose_hiive2 package)

- Launch optim_problems_final.ipynb for the Three Optimization Problems

- Launch optim_neural_net_final.ipynb for the Neural Network's weights Optimization
